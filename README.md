## 项目简介
本项目使用gin、gorm和ssh、sftp开发。旨在编写一个轻量，易用，多平台的运维项目。

现阶段目的是做一个阉割版的xshell并简单的实现ansible或者saltstack的部分功能。
### 查看后端代码请移步到 [oms](https://github.com/ssbeatty/oms)

# 安装依赖
```shell
npm install
```
或者
```shell
yarn add
```

# 1.运行 
```shell
npm run dev
```
```shell
yarn dev
```
# 2.编译
```shell
npm run build
```
```shell
yarn build
```

# 3.注意 
## node 版本 14.17.1 , vite 版本 2.6.14
